# Лабораторная работа №4: Разработка REST API на Node.js

## Описание
Разработка серверной части приложения для управления данными каталога визового центра.

## Функционал API (CRUD)
Сервер поддерживает следующие операции:
- **GET /stocks** — Получение списка всех тарифных зон.
- **GET /stocks/title** — Поиск по названию.
- **GET /stocks/1** — Получение одной зоны по ID.
- **PATCH /stocks/1** — Обновление зоны.
- **DELETE /stocks/1** — Удаление зоны.

## Стек
- Node.js, Express.js.
- Тестирование: Postman.
