# ЛР 3. Простое веб-приложение. Верстка

## Цель 
Знакомство с node, npm, написание простого приложения на JavaScript. В ходе выполнения работы, вам предстоит ознакомиться с кодом реализации простого интерфейса и вывода данных, и затем выполнить задания по варианту.

## Задания

1. Настроить инструменты для работы
2. Узнать, что такое node, npm и package.json
3. Узнать, как работать с html в JS
4. Инициализировать проект
5. Создать главную страницу, подключить bootstrap
6. Построить простую кнопку на JavaScript
7. Структурировать проект
8. Сверстать главную страницу
9. Сверстать страницу продукта

## Дополнительные задания

1. Создать двухстраничное приложение из примера по вариантам.
Вариант состоит из темы и компонента, который необходимо использовать (компонент можно применить по своему усмотрению).
2. Все данные должны соответствовать вашей теме.
